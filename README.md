# Guessing-Game

Begin program
Generate text "Welcome to the number guessing game!"

If users clicks "Yes!"
Begin loop
Else
If user clicks "No ty."
show text "Aw! maybe next time. :("
End program

Start Loop
User attempts = 7
Randomly generate a number from 1 to 100

If user inputs correct randomly generated number
show text "Winner!"
Else
If user inputs incorrect number
attempts = attempts -1
show text "Incorrect! -1 attempt, try again"
Or
If user inputs incorrect number 7 times
show text "Sorry, you lose! Would you like to try again?"

If user clicks "Yes!"
continue loop
Else
If user clicks "No."
Exit loop

End loop when user chooses to not replay
